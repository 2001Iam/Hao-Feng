# 👋 Hi, 我是丰豪 (Feng Hao)

**浙江理工大学在读硕士**

欢迎来到我的主页！我是一名专注于**计算机视觉**领域的算法开发者与研究者。我的核心研究方向是**浑浊水下的图像复原和单目深度估计**。我具备从前沿算法探索（拥有 2 篇中科院二区 SCI 论文，含共同一作）到工业界实际场景落地（海康威视、同盾科技算法实习）的完整闭环能力。

### 🔭 研究与开发方向

* **水下视觉感知与深度估计**：目前正在推进 PRPF (Polarization-RGB Progressive Fusion Framework) 的相关研究，旨在解决浑浊水域中的图像退化和深度预测难题。
* **多模态模型架构创新**：在模型端基于 DINOv2 (Encoder) 与 DPT (Decoder) 探索多模态架构，融合 RGB 与 DOLP 偏振信息增强模型对散射干扰的表征能力，并创新设计了偏振引导适配器 (Polarization-Guided Adapter, PGA)。
* **端侧部署与工程优化**：负责将水下去散射模型与视觉算法部署至 Jetson 平台，利用 TensorRT 与 C++ 结合多线程方案进行显存占用、时延与吞吐性能优化。
* **日常学习与实验**：持续跟进前沿深度估计开源工作（如在 `underwater_fh` 数据集上评估 ZoeDepth，对比 DepthAnythingv2、Unidepthv2 等），并配合实验室开展模型的实地测试与场景验证。

### 🛠️ 核心技术栈

* **编程语言**：C/C++ (熟悉 STL 及常用数据结构与算法)、Python (熟练应用与自动化脚本开发)
* **深度学习框架**：PyTorch (日常核心使用)、TensorFlow、TensorRT (模型转换与推理加速)
* **计算机视觉库**：OpenCV、MMDetection、MMPretrain、Vision Master
* **开发平台与工具**：Linux、Git、PyQt5

### 💼 实践与落地经历

* **算法优化与迭代**：在同盾科技负责二维码反欺诈 20 分类任务，优化 Backbone 选型与输入分辨率，在不增加推理开销的前提下将准确率提升至 95%。
* **工业质检与自动化**：在海康威视基于 Vision Master 开发缺陷检测模型，有效替代部分人工目检，并开发桌面端工具解决批量文件引用的效率问题。
* **农业 CV 应用落地**：在浙大科创中心复用 Re-ID 框架进行目标跟踪，并基于 YOLOv8 与 HSV 算法完成草莓病害检测与成熟度量化。曾带领团队开发面向农户的长势评估微信小程序，成功获批国家级大创项目。
